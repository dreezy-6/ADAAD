The Agents

Three minds. One goal.
Zero coordination.

Each agent is a distinct Claude system prompt with a different cognitive style. They see the same codebase. They never talk to each other. They compete on merit.

🏛️
Architect
Systematic. Long-horizon. Structural.
Focuses on structural coherence, dependency cleanup, and interface contracts. Thinks in systems, not lines. Sees how a change ripples through the whole codebase. Slow wins, clean architecture, lasting improvements.
💭
Dream
Creative. Lateral. Non-obvious.
Explores novel rewrites and experimental approaches. Finds the solution nobody thought to try. Lateral thinking produces proposals that neither Beast nor Architect would generate.
🐉
Beast
Aggressive. Performance-first. Raw.
Hunts for throughput gains, complexity reduction, raw optimization. Doesn't care about elegance — cares about speed. The GovernanceGate stops Beast from going too far. That's the point.

The GovernanceGate

16 rules. In strict order.
One failure = full stop.

The GovernanceGate is the only surface that can approve a mutation. It cannot be overridden by any agent, operator, configuration flag, or pricing tier. This is an architectural invariant, not a policy.
