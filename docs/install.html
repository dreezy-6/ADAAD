📌 About sideloading

ADAAD is distributed outside the Play Store — entirely free. Android calls this sideloading. You download the APK file and install it directly. Android will ask you to allow installations from your browser once; after that it's a normal app. All builds are signed with the InnovativeAI LLC certificate so your device can verify integrity.
