## What's new

### AI Mutation Pipeline (Principal/Staff grade)

Six production subsystems delivering the first self-improving mutation loop:

**`AIMutationProposer`** — Claude API (`claude-sonnet-4-20250514`) connected to the mutation pipeline for the first time. Three agent personas (Architect, Dream, Beast) with engineered system prompts. Pure `urllib.request` — zero third-party deps. Markdown fence stripping for API robustness.

**`EvolutionLoop`** — Five-phase epoch orchestrator: Strategy → Propose → Seed → Evolve → Adapt → Record. `EpochResult` dataclass with 8 fields. `simulate_outcomes=True` for unit-test isolation.

**`WeightAdaptor`** — Momentum-based coordinate descent (`LR=0.05`, `momentum=0.85`) on `gain_weight` and `coverage_weight`. Bounds: `[0.05, 0.70]`. EMA prediction accuracy tracking. JSON persistence after every epoch.

**`FitnessLandscape`** — Persistent per-mutation-type win/loss ledger. Plateau detection (≥3 attempts below 20% → Dream). Agent recommendation decision tree. UCB1/Thompson Sampling extension point (Phase 2).

**`PopulationManager`** — GA-style: BLX-alpha=0.5 crossover, `MAX_POPULATION=12`, `ELITE_SIZE=3`, `CROSSOVER_RATE=0.4`. MD5 fingerprint deduplication. Crossover children inherit `parent_id` for elitism bonus eligibility.

**`MutationScaffold v2`** — `ScoringWeights` dataclass, `PopulationState`, five new lineage fields on `MutationCandidate` (100% backward-compatible). Adaptive acceptance threshold.

## Upgrade path

No migration required. Activate AI mutations by setting `ADAAD_CLAUDE_API_KEY` and running:

```python
from runtime.evolution.evolution_loop import EvolutionLoop
loop = EvolutionLoop(api_key=os.environ["ADAAD_CLAUDE_API_KEY"], generations=3)
result = loop.run_epoch(context)
```

Without `ADAAD_CLAUDE_API_KEY`, the system runs in simulation mode (`simulate_outcomes=True`).
