# ADAAD Brand Assets

Optimized SVG assets for README/documentation rendering on GitHub are stored in this folder.

- `adaad-brand-identity.svg`: primary ADAAD identity hero visual.
- `adaad-governance-model.svg`: compact governance lifecycle model.
- `aponi-context.svg`: Aponi authoring boundary and governance authority context.

Original governance-focused diagrams remain in `docs/assets/` (for example `governance-flow.svg` and `adaad-banner.svg`) to preserve existing references and source visuals.
