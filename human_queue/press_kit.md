## One-sentence description

ADAAD is the only AI coding system that can prove — cryptographically — exactly what it changed and why.

## One-paragraph description

ADAAD runs three Claude-powered AI agents in a continuous loop against your codebase. Every proposed mutation passes a 16-rule constitutional governance gate before a single byte changes. Approved mutations are SHA-256 hash-chained into an append-only evidence ledger and are deterministically replayable months later. Unlike GitHub Copilot or Cursor, ADAAD produces a legally auditable, cryptographically provable record of every autonomous code decision. Free Community tier forever (MIT, self-hosted).
