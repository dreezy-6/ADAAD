**Body text to paste:**

Three Claude-powered AI agents (Architect, Dream, Beast) continuously propose code mutations. Each proposal is scored by a genetic algorithm, then must pass a 16-rule constitutional policy engine before a single byte changes.

What makes this different from Copilot/Cursor:

- **Deterministic replay**: every decision can be re-run byte-identical. Divergence halts the pipeline.
- **Constitutional gating**: 16 hard rules evaluated per mutation. One failure = full halt. No exceptions.
- **SHA-256 hash-chained evidence ledger**: every governed step is permanently audited.
- **Self-calibrating weights**: scoring adapts via momentum gradient descent across epochs.
- **Genetic population**: candidates cross, mutate, and compete. The fittest survive to the governance gate.
- **Roadmap self-amendment**: the engine proposes changes to its own roadmap. Humans approve. No auto-merge path exists by architectural invariant.

The governance model is architectural, not configural. The `GovernanceGate` is the only surface that can approve mutations — this cannot be bypassed by any agent, operator, or configuration flag.
